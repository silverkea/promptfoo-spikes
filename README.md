# promptfoo-spikes

A repository for spikes relating to the discovery of [promptfoo](https://www.promptfoo.dev/) capabilities.

## About

[promptfoo](https://github.com/promptfoo/promptfoo) is an open-source tool for testing and evaluating LLM prompts and models. This repository contains exploratory spikes to understand and experiment with its features, including:

- Prompt evaluation and testing
- Model comparison
- CI/CD integration for LLM quality checks
- Red-teaming and safety testing

## Structure

Each spike is self-contained and focused on a specific promptfoo capability or use case.
